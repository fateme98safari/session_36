# Image Processing-HSV


simple projects using opencv with python.




## python

This project is written in Python version 3.10.8

I have used the [opencv](https://pypi.org/project/opencv-python/) and [Numpy](https://numpy.org/) 








### **How to install**

 ```

 ```
